# Lootsy Web 3.1

Nyheter:
- Redirect-API `/api/redirect` med klickloggning
- Deal-detaljsidor `/deal/[id]` med SEO + JSON-LD
- Sidor: /about, /contact, /privacy
- sitemap.xml

Snabbstart:
1) `cp .env.example .env.local` och fyll i nycklar
2) `npm i && npm run dev`
3) Öppna `/api/sync` för att seed:a deals
